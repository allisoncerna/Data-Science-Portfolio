# Data-Science-Portfolio
Data Science &amp; Analytics Portfolio | End-to-end projects featuring automated data ingestion (Kaggle API), financial time-series forecasting in Python, and corporate profitability analysis using SQL and Tableau

- Project 1: Personal Finance Optimizer & Budget Forecasting 

        Tech Stack: Python (Pandas, Matplotlib, Seaborn), Kaggle API, Time-Series Models.

        The Goal: Automate the ingestion of 32,000+ financial records to identify "spending leaks" and forecast future discretionary income.

        Key Insight: Built a data pipeline to convert raw CSV data into actionable savings patterns, focusing on personal finance optimization.

- Project 2: Corporate Financial Health & Profitability SQL Analysis

        Tech Stack: SQL (PostgreSQL/MySQL), Tableau, Data Modeling.

        The Goal: Analyze large-scale corporate datasets to identify high-margin business segments and optimize profit structures.

        Key Insight: Created a dashboard that visualizes the relationship between operating costs and revenue, providing a clear "story" for executive decision-making.
